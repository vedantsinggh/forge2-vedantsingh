# FastAPI Engineering Skill

## Overview
This skill covers the development of production-ready FastAPI applications using Python. It includes building asynchronous endpoints, structured routing, dependency injection, and Pydantic validation.

## Capabilities
- Structuring APIs with APIRouter.
- Asynchronous database operations using SQLAlchemy or Tortoise.
- Setting up request validation and response serialization using Pydantic v2.
- Designing custom middleware, exception handlers, and security dependencies.
- Integrating lifespan events for startup and shutdown procedures.
