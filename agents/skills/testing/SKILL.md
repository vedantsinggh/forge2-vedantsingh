# Testing and Validation Skill

## Overview
This skill focuses on writing robust test suites using `pytest`, configuring mocks, asserting outputs, and tracking test coverage.

## Capabilities
- Structuring test files and folders (`tests/test_*.py`).
- Creating reusable `pytest.fixture` functions.
- Mocking external services, HTTP requests, and databases.
- Measuring and reporting code coverage.
- Writing unit, integration, and end-to-end tests.
