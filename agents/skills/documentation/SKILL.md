# Documentation Skill

## Overview
This skill focuses on automatically generating and updating clean, structured, and informative markdown files (README, Architecture, API reference, Changelogs) for software systems.

## Capabilities
- Structuring README files with clear installation, setup, and usage guides.
- Creating ARCHITECTURE.md documents detailing systems flow, directory mappings, and Mermaid flowcharts.
- Generating API.md detailing endpoints, payloads, and response status codes.
- Creating SPRINT_REPORT.md logs representing work cycles, security reports, and QA summaries.
