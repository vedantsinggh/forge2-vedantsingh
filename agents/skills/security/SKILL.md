# Security Engineering Skill

## Overview
This skill focuses on scanning for vulnerabilities, ensuring safe coding practices, and preventing risks like SQL Injection, XSS, CSRF, and hardcoded secrets.

## Capabilities
- Executing static analysis security tools (Bandit, Semgrep, pip-audit).
- Identifying secret exposures in files and logs.
- Reviewing OWASP Top 10 vulnerabilities (SQL Injection, CSRF, XSS, Path Traversal).
- Secure credential management and dotenv setups.
- Safe subprocess usage to prevent command injection.
